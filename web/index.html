WebAssembly-powered cheat detection for Super Smash Bros Melee

About

This tool analyzes Slippi replay files (.SLP) for evidence of cheating in Super Smash Bros Melee. All analysis happens locally in your browser using WebAssembly - no data is uploaded to any server.

Checks Performed:

  • Box Travel Time - Detects digital box controllers by measuring travel coordinate patterns
  • Disallowed C-Stick Values - Flags illegal C-stick coordinates indicating hardware manipulation
  • Uptilt Rounding - Detects artificial rounding on analog controllers
  • Fast Crouch Uptilt - Detects impossibly fast state transitions (<3 frames)
  • Illegal SDI - Detects impossible SDI (Super Defensive Input) patterns
  • GoomWave Clamping - Detects GoomWave hardware modifications
